# ConvoDesk – Project Contract

## Product
ConvoDesk is an AI-powered WhatsApp CRM for managing customer conversations, leads, follow-ups, and analytics.

## Tech Stack
Frontend:
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Recharts
- TanStack Query
- Zod
- Framer Motion (optional)

Backend:
- FastAPI
- SQLAlchemy
- Pydantic

Infra:
- Supabase Postgres
- Supabase Auth

AI:
- Gemini API

## Core Modules
- Dashboard
- Inbox
- Contacts
- Pipeline
- Follow-ups
- Analytics
- Settings

## Main Entities
- Contact
- Conversation
- Message
- FollowUp
- Note
- Tag
- Agent

## UI Style
- Modern SaaS CRM dashboard
- WhatsApp-inspired green accent
- Clean cards, tables, kanban board
- 3-column inbox layout

## Hard Rules
- Do not change stack without explicit instruction
- Do not rename routes/folders unless requested
- Do not replace Supabase with Firebase
- Do not rewrite entire pages when only one component is requested
- Preserve TypeScript types and API contracts
